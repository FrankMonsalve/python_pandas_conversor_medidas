# Conversor de medidas
Realice un programa en Python con Pandas y OpenPyXl en el cual leemos un archivo Excel y convertimos las unidades de centimetros en pulgadas

Para poder ejecutar este proyecto despues de clonarlo debes hacer en la carperta del proyecto la siguiente instruccion:

```
pip install -r requirements.txt
```

Una vez hecho esto para que convierta el excel de centimetros a pulgadas, debemos ejecutar en consola

```
python programa_conversor.py
```
